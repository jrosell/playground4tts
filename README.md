# playground4tts

In this repo there are multiple Text-to-Speach models available in catalan:

* projecte-aina/matxa-tts-cat-multiaccent
* facebook/mms-tts-cat
* wetdog/speecht5_tts_commonvoice_ca
* wetdog/speecht5_tts_openslr_ca

## Install

```
bash install.sh
```
